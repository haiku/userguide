Lettore Midi

Deskbar:Applicazioni
Percorso:/boot/system/apps/MidiPlayer
Impostazioni:~/config/settings/MidiPlayerSettings


Come il nome suggerisce, il Lettore Midi è utilizzato per eseguire file musicali in formato MIDI. I file MIDI sono speciali, in quanto non contengono audio in forma digitalizzata e codificata, ma solo una descrizione: tieni questa nota per questo tempo, con questo volume e suonala con lo strumento X.
Per quanto questo consenta di avere file di dimensioni molto contenute, la qualità dipende molto dalla libreria di strumenti a disposizione (i cosiddetti "SoundFont"). I SoundFont, non a caso, possono avere dimensioni considerevoli, dipendenti della quantità e della qualità dei campioni in essi contenuti.

Haiku include una libreria SoundFont piccola, ma di qualità dignitosamente alta. Gli utenti che usano i file MIDI in modo più serio probabilmente hanno già a disposizione delle librerie di SoundFont di qualità molto più alta.

Per usare altri SoundFonts (che non provengono da HaikuDepot o in un pacchetto in formato HPKG), copia semplicemente il file .sf in ~/config/non-packaged/data/synth/. Puoi scegliere il banco attivo attraverso il pannello di preferenze Media sotto MIDI.

L'interfaccia di Lettore Midi è molto semplice. Fai doppio click oppure trascina un file midi sulla finestra principale per iniziare la riproduzione. Puoi controllare il volume con l'apposito slider e aggiungere un effetto di riverbero dal menu Riverbero. La spunta Oscilloscopio mostrerà una visualizzazione nella forma di un'oscilloscopio nell'area in alto.
