VirtualMemory

Deskbar:Preferensi
Lokasi:/boot/system/preferences/VirtualMemory
Pengaturan:~/config/settings/kernel/drivers/virtual_memory
~/config/settings/VM_data - Stores the panel's window position.

Memori virtual memungkinkan sistem menukar memori ke harddisk, jika RAM dapat digunakan lebih masuk akal untuk hal-hal lain. Jadi, bahkan jika Anda memiliki banyak RAM, menyediakan memori virtual tidak pernah merupakan ide yang buruk.

Biasanya, file swap ditulis ke partisi boot. Jika Anda kehabisan memori bebas di sana, Anda dapat menonaktifkan manajemen swap otomatis dan memilih partisi lain yang terpasang dari menu popup. Ukuran file swap diatur dengan slider di bawah ini.
Jika Anda sering mengalami hard disk bermasalah karena sistem memori virtual menukar memori masuk dan keluar, Anda dapat mencoba menggunakan harddisk terpisah untuk menukar file untuk menghindari kemacetan I/O. Cukup partisi lain pada harddisk yang sama dengan sistem / data Anda tidak akan membantu. Meningkatkan RAM Anda tentu saja merupakan cara paling efektif...

Bawaan mengatur ulang semuanya ke nilai bawaan.
Pulihkan mengembalikan pengaturan yang aktif ketika Anda memulai preferensi VirtualMemory.
